User Guide - Sigma

Introduction

Welcome to Sigma, the app that helps you achieve your fitness goals! This user guide will walk you through the various features and functionalities of Sigma so that you can get the most out of the app.

Getting Started

To get started with Sigma, simply download the app from the App Store or Google Play and create an account. Once you have created an account, you can start using the app to track your workouts, set goals, and more.

Tracking Workouts

Sigma makes it easy to track your workouts. Simply go to the "Workouts" tab and select "New Workout" to log a new workout. You can then enter details such as the type of exercise, the duration, and the intensity. Sigma will automatically track your progress over time, so you can see how far you have come.

Setting Goals

One of the key features of Sigma is the ability to set goals for yourself. Whether you want to lose weight, build muscle, or simply stay in shape, Sigma can help you achieve your goals. Simply go to the "Goals" tab and select "New Goal" to create a new goal. You can then set parameters such as the target weight or body fat percentage, and Sigma will provide you with a customized plan to help you achieve your goal.

Conclusion

Thank you for using Sigma. We hope this user guide has been helpful in getting you started with the app. If you have any questions or feedback, please don't hesitate to contact us.

Happy training!
